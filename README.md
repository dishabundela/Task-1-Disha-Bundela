# 🚢 Titanic Dataset Analysis Using Python

##  Overview
This project focuses on analyzing the Titanic dataset using Python. The dataset was cleaned, processed, and analyzed to identify survival patterns and passenger demographics.

##  Objectives
- Load and understand the Titanic dataset
- Handle missing values
- Perform data cleaning
- Analyze passenger information
- Visualize important insights

##  Technologies Used
- Python
- Pandas
- Matplotlib
- VS Code

##  Dataset
Titanic-Dataset.csv

##  Data Cleaning
- Filled missing Age values using mean
- Filled missing Embarked values using mode
- Handled Cabin missing values

##  Analysis Performed
### Survival Analysis
- Survived: 342
- Not Survived: 549

### Gender Distribution
- Male: 577
- Female: 314

### Passenger Class Distribution
- 1st Class: 216
- 2nd Class: 184
- 3rd Class: 491

##  Visualizations
- Survival Count Graph
- Gender Count Graph
- Passenger Class Graph

##  Key Findings
- Most passengers did not survive.
- Majority of passengers were male.
- Most passengers traveled in 3rd class.
